# CV Portfolio

Portfolio personal desarrollado como proyecto principal para **mostrar mi CV, habilidades y trabajos destacados**.  
Construido en **React** con **Tailwind CSS** y animaciones suaves usando **Framer Motion**, incluye un **diseño responsivo**, formulario de contacto funcional y enlaces a redes profesionales.

## Tecnologías
- **React**  
- **Tailwind CSS**  
- **Framer Motion**  
- **Formspree** (formulario de contacto)  
- **Vite** (herramienta de build)

## Características principales
- Secciones: Home, Proyectos, Sobre mí, Contacto  
- Componentes reutilizables para UI y formularios  
- Formulario de contacto con validación y protección anti-spam  
- Animaciones y diseño responsive para mejor UX

## Proyectos destacados
Dentro de este portfolio se muestran algunos de mis proyectos más relevantes:  

- **SoulBeat (Full Stack)** – E-commerce de auriculares con frontend en React/Tailwind y backend en Node.js/Express/MongoDB. Incluye panel de administración, favoritos, carrito y tema claro/oscuro.  
- **Dashboard de países** – Dashboard interactivo con CRUD completo usando Node.js, Express y EJS, para explorar y gestionar información de países.  
- **Mandala Cakes** – E-commerce de pastelería desarrollado en Bubble.io, centrado en UX, diseño de producto y experiencia de usuario.
